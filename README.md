# -Indicadores-de-inter-s-CCMRA---Antofagasta-
Repositorio indicadores de interés para la 'Corporación Clúster Minero de la Región de Antofagasta'. Datos extraidos desde la API del Banco Central de Chile 'Base de Datos Estadísticos (BDE).
El siguiente repositorio almacena datos e información de interés y relevancia para la 'Corporación Clúster Minero de la Región de Antofagasta'. Estos datos son extraídos de la API para Base de Datos Estadísticos del 'Banco Central de Chile'
El Banco Central de Chile dispone un servicio web que permite buscar y extraer información económica contenida en su base de datos.
